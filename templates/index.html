Disclaimer
This website is not an official source of information but rather an exercise of data visualization UX that tries to fill the gaps of the oficial reporting.
Note: as of 13/03/2022 our health authority (DGS) reduced the scope of reporting, preventing the geospatial and age-stratified plots from being updated.

Notes
Data is presented as a 7 day moving average to make reading the plots easier. The goal of this dashboard is allowing the visualization of mid term trends rather than the short term noise.

Credits
The source code for this application can be found here.
The web frontend loads in Portable Web App (PWA) mode thanks to the contribution of @pmac-dev.
The static content was adapted from HTML5UP.
The plots are made with Bokeh.
Historical data was obtained from the DSSG-PT repositories and current data is merged from DGS and DSSG-PT data sources.
